# docs

SweetRiceMC 服务器帮助文档

# 贡献

欢迎各位通过 Pull Requests 提交或修改文档，大致流程如下：

* 登录 Github 账号
* Fork 本仓库
* 作出修改
* 通过 Pull Requests 提交

如果你不会操作，也可以将需要提交的文章发送到邮箱 mrxiaom@qq.com，或者发送到 Issues。

文章均放置在 [docs](docs) 文件夹中，添加文章后请顺手将文章添加到 [sidebar.js](sidebar.js)。
