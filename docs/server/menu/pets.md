# 幻兽界面

![图片]()

打开菜单，鼠标左键点击 **“宠物”** 按钮即可进入幻兽界面。

幻兽界面会显示你拥有的幻兽，**鼠标左键点击对应头像即可召唤对应幻兽**{.underline .red}。

![幻兽界面]()

::: tip 幻兽 迅猛龙
![幻兽迅猛龙]()
:::

# 幻兽的功能

目前幻兽的功能有限，仅 **部分幻兽具有骑乘功能**{.underline .red}。

当然，幻兽本身也是一种实力的象征，带着幻兽去尽情炫耀吧。

![玩家骑在迅猛龙上]()

# 获取幻兽

目前幻兽仅可通过在[幸运神殿](worlds/crate.md)中使用***幻兽契约***{.rainbow}打开幻兽宝箱获得。
