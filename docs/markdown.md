## 表格
表格分为三部分，表头、排列方式、主体。

如下所示，我们用 `|` 来分隔每一列。  
表头下面一行就是排列方式，用三个或以上的 `-` 加上 `:` (可选) 来表示。
+ `|---|` 表示单元格内容居左 (默认)
+ `|:---|` 表示单元格内容居左
+ `|---:|` 表示单元格内容居右
+ `|:---:|` 表示单元格内容居中

为了源代码美观，排列方式通常会添加多个 `-` 使其长度与单元格内容一致，表头添加多个空格同理。

如果需要在单元格内换行，请使用 html 代码 `<br>`

```markdown
| 表头文字 | 表头文字 | 表头文字 | 表头文字 |
| --- |:--- |:---:| ---:|
| 默认 | 居左 | 居中 | 居右 |
| 单元格内容<br>换行 | 单元格内容 | 单元格内容 | 单元格内容 |
```

| 表头文字 | 表头文字 | 表头文字 | 表头文字 |
| --- |:--- |:---:| ---:|
| 默认 | 居左 | 居中 | 居右 |
| 单元格内容<br>换行 | 单元格内容 | 单元格内容 | 单元格内容 |
